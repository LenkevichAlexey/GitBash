1) Посмотреть где я. - pwd
2) Создать папку. - mkdir papka
3) Зайти в папку. - cd papka
4) Создать 3 папки. - mkdir {papka2,papka3,papka4}
5) Зайти в любоую папку. - cd papka2
6) Создать 5 файлов (3 txt, 2 json). - touch text.txt text2.txt text3.txt file.json file1.json
7) Создать 3 папки. - mkdir papka5 papka6 papka7
8. Вывести список содержимого папки. - ls -la
9) + Открыть любой txt файл. - vim text.txt
10) + написать туда что-нибудь, любой текст. - Нажать I, которая переключает в режим insert - редактирования
11) + сохранить и выйти. - нажать Esc, а затем ввести :wq + enter
12) Выйти из папки на уровень выше. - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv {text.txt, text2.txt} papka5
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp {file.json, file.json} papka6
15) Найти файл по имени. - find papka5 -name text.txt / find . -name file.json
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. tail -f papka/papka5/text.txt
17) вывести несколько первых строк из текстового файла. - head text.txt / head n- 4 text.txt
18) вывести несколько последних строк из текстового файла. - tail text.txt / tail -n 2 text.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less text.txt, чтобы выйти из файла ввести :q
20) вывести дату и время. - date +"%H:%M %d/%m/%Y"
